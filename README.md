Для Использования приложения необходимо:
1. в настройках используемого gmail аккаунта разрешить использования POP/IMAP
2. основной язык установить "ENGLISH (US)" 
3. В настройках профиля в разделе "Безопасность и вход" в подразделе "Вход в аккаунт Google" установить значение флажка "Ненадежные приложения разрешены" в положение "On" (включено)
4. Скопировать содержимое репозитория и запустить файл install.php
5. Пройти установку
6. Зайти в index.php ввести данные для входа в аккаунт.
